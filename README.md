# Solidity Smart Contract Collection

This repository contains a collection of Solidity smart contracts demonstrating core Web3 concepts and security patterns.

## Features Implemented
- **DeFi Primitives**: Simple Banking system with deposit/withdraw logic (`SimpleBank.sol`).
- **DEX Logic**: Uniswap V2 style Pair creation using `CREATE2` opcode (`UniswapPairFactory.sol`).
- **Inter-contract Communication**: Examples of `call`, `delegatecall`, and `interface` interactions.
- **Error Handling**: Robust error management using `try/catch` and custom errors.

## Tech Stack
- Solidity ^0.8.21
- OpenZeppelin Standards (Referenced)
